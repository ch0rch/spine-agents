# Product Knowledge

## Instrucciones

Leé los siguientes archivos para conocer el producto o servicio:

1. `areas/producto/overview.md` — Qué es, cómo funciona, diferenciadores
2. `areas/producto/estrategia.md` — Roadmap y dirección del producto

## Qué encontrás en este contexto

- Descripción del producto o servicio
- Cómo funciona (flujo, pasos, mecánica)
- Diferenciadores frente a la competencia
- Casos de uso principales
- Prueba social y métricas de producto

## Cómo aplicar

- Usá esta skill cuando el contenido o análisis mencione características del producto
- No inventes features o datos que no estén documentados
- Si hay info desactualizada, activá `@curator` para actualizarla

## Completar con información de tu empresa

Este archivo toma información de `areas/producto/`. Completá esa carpeta con la información real de tu producto antes de usar esta skill en producción.
