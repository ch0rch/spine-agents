# Inbox

Zona de trabajo de los agentes. Los outputs, borradores y archivos generados por agentes van acá.

## Cómo funciona

1. Un agente genera un output (documento, reporte, análisis, borrador, etc.)
2. Lo guarda en esta carpeta
3. El equipo humano revisa
4. Si está bien, lo mueve a la ubicación final o lo usa directamente
5. Si necesita ajustes, se le pide al agente que lo modifique

## Convención de nombres

```
YYYY-MM-DD-agente-descripcion.md
```

Ejemplo: `2026-03-15-marketing-propuesta-campaña-verano.md`

## Limpieza

Mover o eliminar archivos procesados periódicamente para mantener el inbox limpio.
